THOR Challenge

Navigating towards a set of objects in indoor scenes based only on visual input.

Introduction

In this challenge, the task is to navigate towards a set of objects in indoor scenes based only on visual input. In [1], we introduced AI2-THOR, which is a framework that integrates a physics engine with a deep learning framework in a photo-realistic setting. AI2-THOR enables navigation in a set of near-photo-realistic indoor scenes, and we will use this framework for the challenge. The agents that the challenge participants provide receive visual input from THOR and should perform an action based on the visual input. We will hold the challenge in two settings: (1) the participants will have access to the test scenes, (2) the agents will be evaluated on totally unseen scenes.
